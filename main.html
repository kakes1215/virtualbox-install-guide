Getting Started

Follow the instructions below to download the software you need to get started

  1. Start by going to the VirtualBox website with the following link: https://www.virtualbox.org/wiki/Downloads
  2. Download the Platform package and the Extension Pack
  3. Start VirtualBox and add the Extension Pack by going to File, Preferences and then Extensions
  4. For more detailed instructions on how to install and setup VirtualBox use the following link: https://www.virtualbox.org/manual/ch01.html#intro-installing
  5. Go to the Ubuntu website to download the Ubuntu Desktop OS:https://ubuntu.com/download
  6. The download should be a .iso file, which VirtualBox will read and create the machine

Create New Virtual Machine

We will now create your very first virtual machine.

  1. With VirtualBox open click the New button to create a new virtual machine
  2. Give your virtual machine a name and then select Linux for the Type and Ubuntu 64-bit for the Version
  3. Click Next
  4. We will now put in how much RAM this virtual machine will use. Ubuntu does not need a lot of RAM to run smoothly so this will completely depend on what you are using this virtual machine for and your computer. For example, if you have 8 GB of RAM on your computer then allocate 2GB (2048M) to the VM.
  5. Click Next
  6. To setup the hard drive select Create a Hard Drive Now then click Create
  7. Select VDI then click Next
  8. We want this drive to be Dynamically Allocated which means the disk space is not used until the virtual machine needs it
  9. Click Next
  10. Now choose the size for the new drive. For this tutorial, you should probably give yourself 20GB
  11. Click Create
  12. Go to System and then Processor and reserve CPU cores for your virtual machine. If you have at least 4 CPU cores then reserve 2 for the VM

Install Ubuntu

We are now going to use the .iso file you downloaded in the Getting Started section to install Ubuntu on your empty virtual machine

  1. Click the Settings button
  2. Click the small blue circle icon to the right of first storage device (it should say Adds optical drive when you hover over it)
  3. Click Choose Disk and then Add. Find your .iso file and select it
  4. Click OK
  5. You should now see your .iso file listed under the storage device. Remove any other empty devices if applicable
  6. Return to the main VirtualBox window and click the Start button
  7. Follow the prompts to install Ubuntu. If you need more instructions on how to do this use the following link: https://tutorials.ubuntu.com/tutorial/tutorial-install-ubuntu-desktop#0

    8. *NOTE: Do not worry about warnings of erasing your whole disk. The VM only knows about the drive you created and not your actual computer drive.

  8. After Ubuntu finishes installing, let it reboot and hit Enter if prompted to do so
  9. Your Ubuntu installation is now up and running.
  10. Login into your Ubuntu installation and open a Terminal from the launch menu (lower left corner).
  11. Type in the following two commands:
    12.             
                sudo apt-get update
                sudo apt-get upgrade
  12. sudo is a Linux command that will run the command with administrator privileges and will require your password.
  13. apt-get is a Linux program that will update software or install/uninstall packages
  14. So the above commands will first get the information about what updates are available and then update your computer based on that information

Install Software

We are now going to install software that you may need for you Ubuntu machine

C++ Compiler

Use the following command: 

sudo apt-get install g++

Verify it is installed correctly by typing in the following:

g++ --version

Make sure it responds with an identifying message making clear that the software is installed and running

Emacs

Use the following command: 

sudo apt-get install emacs

Gnu Plot

Use the following command: 

sudo apt-get install gnuplot

Verify it is installed correctly by typing in the following:

which gnuplot

Make sure it responds with an identifying message making clear that the software is installed and running

SQL Developer

  1. Download a JDK here
  2. Install the package with the following command: 
    3. rpm -Uhv sqldeveloper-(build number)-1.noarch.rpm
  3. Go to the sqldeveloper directory:
    4. cd sqldeveloper
  4. Run the sqldeveloper file
    5. ./sqldeveloper.sh
  5. Put in the path to the JDK that you downloaded at the beginning
  6. SQL Developer will now launch automatically.
